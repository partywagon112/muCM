# μCM - Micro Control Module

A small ESP32 based flexible controller, intended for Matter to control an RGB light or motor, but very very flexible. Based around ESP32-C6 for flexibility. Includes a 3V3 CAN driver for interacting on small networks, two pairs of half bridges allow driving servos, LEDs, or outputs more generally.
